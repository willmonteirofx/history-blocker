# HistoryBlocker Chrome Extension

Extensão do Chrome que impede o navegador de salvar histórico para sites contendo palavras-chave específicas na URL ou título.

## Funcionalidades

- Bloqueia automaticamente histórico para sites com palavras-chave especificadas
- Proteção por PIN para acessar configurações
- Toggle para ativar/desativar a extensão
- Opção de ocultar/mostrar palavras-chave visualmente

## Instalação

1. Abra o Chrome e vá em `chrome://extensions/`
2. Ative o "Modo do desenvolvedor"
3. Clique em "Carregar sem compactação" e selecione a pasta da extensão

## Como Usar

1. Clique com botão direito no ícone da extensão e selecione "Opções"
2. Defina um PIN na primeira vez (4-10 dígitos)
3. Adicione palavras-chave que deseja bloquear
4. Use o ícone da extensão para ativar/desativar rapidamente

## Notas

- Palavras-chave são verificadas na URL e no título da página
- Todas as configurações são armazenadas localmente
- Se esquecer o PIN, limpe os dados da extensão em `chrome://extensions/`
